Proyecto Final

Studying Time

Como proyecto final quisimos hacer una aplicación que nos ayudara a gestionar nuestro tiempo de estudio, una aplicación en la que determinaremos un tiempo para estudiar y uno para descansar, es decir, una aplicación similar a emplear la técnica POMODORO.

La técnica POMODORO consiste en determinar un intervalo de tiempo para estudiar y otro para descansar, el cual generalmente es de 25 minutos para estudiar y 5 para descansar.

Como hicimos la aplicación

Conseguimos hacer la aplicación con ayuda de MIT APP INVENTOR 2 y sus sensores, esta app facilita muchísimo el hacer aplicaciones para el sistema operativo Android, es por esto que la escogimos, además de por la gran cantidad de funciones que tiene implementada, como los son los sensores, de estos utilizamos el acelerómetro, para dar inicio al temporizador, y utilizamos el sensor de orientación, para darle movimiento a la imagen y hacer una app un poco más interactiva y divertida.

Como creamos la pagina web

Para crear la página utilizamos una plantilla de HTML5 UP, luego de descarga y descomprimirla, modificamos el código con la ayuda del editor Sublime text, y logramos publicar la página gracias a GITHUB PAGE, ya que subimos los archivos de la propia página a un repositorio en GITHUB.
